# Compliance Documentation

This folder contains templates and checklists for GDPR and privacy compliance work on Vibeverk customer deployments.

**Important:** All templates in this folder are starting points that require customer-specific confirmation and qualified legal review before use. They are not legal approval and do not constitute legal advice. Each template must be completed with actual customer-specific facts before it can be used for compliance purposes.

| File | Description |
|---|---|
| [data-map-template.md](data-map-template.md) | Reusable template for mapping data flows per customer deployment |
| [customer-go-live-checklist.md](customer-go-live-checklist.md) | GDPR/compliance checklist to complete before a customer site goes live |
